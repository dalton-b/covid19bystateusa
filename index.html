COVID-19 Reported New Cases and Deaths by US State

The blue graphs show the reported number of new cases of COVID-19 in the last 28 days in US States and territories over time. Keep in mind that these are only reported cases, and the actual number of cases may be much higher.

The orange graphs show an 8 day moving average of the number of deaths, with the actual number of deaths on the same graph in light blue. This averaging is done because the death numbers are highly variable from day to day, so averaging them makes the graph more readable.
